## 📖 Описание

**TestingSystem** — это веб-приложение для образовательных учреждений, позволяющее:

- ‍🏫 **Преподавателям** создавать тесты, вопросы и управлять дисциплинами
- 👨‍🎓 **Студентам** проходить тесты и видеть результаты
- 👨‍💼 **Администраторам** управлять пользователями и ролями

---

## ✨ Возможности

### Для администратора
- ✅ Управление пользователями и ролями
- ✅ Просмотр статистики по тестам
- ✅ Управление дисциплинами

- ✅ Создание и редактирование дисциплин

- ✅ Настройка времени и проходного балла

- ✅ Прохождение доступных тестов
- ✅ Ограничение по времени
- ✅ Мгновенный результат после завершения
- ✅ История пройденных тестов

### Типы вопросов
-  Один правильный ответ
- ☑️ Несколько правильных ответов
- 📝 Свободный текстовый ответ

## 🚀 Установка и запуск

### Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- [Git](https://git-scm.com/)
- 4 GB+ свободной памяти

### Быстрый старт

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/USERNAME/TestingSystem.git
   cd TestingSystem
   ```
2. Запуск проекта
    ```bash
    docker-compose up --build -d
    ```
3. Проект находится по адресу: http://localhost:8080
4. Остановка проекта:
   ```bash
   docker-compose down
   ```

##🎯 Использование

### Создание теста (для преподавателя)

1. Войдите под учётной записью преподавателя
2. Перейдите в раздел **«Дисциплины»**
3. Выберите дисциплину или создайте новую
4. Нажмите **«Создать тест»**
5. Заполните параметры:
   - Название и описание
   - Ограничение по времени (в минутах)
   - Проходной балл (в %)
6. Добавьте вопросы и варианты ответов
7. Опубликуйте тест

### Прохождение теста (для студента)

1. Войдите под учётной записью студента
2. Выберите доступный тест
3. Ответьте на вопросы в отведённое время
4. Нажмите **«Завершить тест»**
5. Просмотрите результат
